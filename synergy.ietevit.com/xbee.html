Xbee Communication Workshop





Description


Xbee Communication workshop mainly focuses on the students eager to learn Wireless Communication using Xbee Protocol and AVR microcontroller. Xbee transiver module is easy to integrate and provide reliable wireless communication upto 100 meters. Xbee supports point to point and multi point networks

23rd - 24th Jan'16
